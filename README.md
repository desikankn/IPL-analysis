# IPL Analytics Dashboard using Power BI

## Project Overview
This project is an interactive IPL (Indian Premier League) Analytics Dashboard developed using Power BI. The dashboard transforms raw cricket data into meaningful insights through visual analytics, KPI tracking, and performance comparisons.

The project helps analyze team performance, player statistics, venue trends, and season-wise IPL data using dynamic visualizations and advanced DAX calculations.

---

## Features
- Team performance analysis
- Player statistics and comparisons
- Orange Cap & Purple Cap insights
- Season-wise IPL trends
- Venue-based match analysis
- Interactive filters and slicers
- KPI dashboards and visual storytelling
- Advanced DAX measures and calculations

---

## Technologies Used
- Power BI
- DAX
- Power Query
- Data Modeling
- Excel / CSV Dataset

---

## Dashboard Insights
- Win/Loss analysis of IPL teams
- Top-performing batsmen and bowlers
- Strike rate and economy analysis
- Match trends across seasons
- Venue impact on team performance
